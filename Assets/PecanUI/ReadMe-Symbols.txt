# PECAN_NAVIGATOR
Enable navigator for harward controller (ex, TV's remote)

# PECAN_ENABLE_IMPRESSION
Force enable goPlay impression ads

# PECAN_DISABLE_IMPRESSION
Force disable goPlay impression ads

# PECAN_ENABLE_COUPON_REWARD
Force enable goPlay coupon reward

# PECAN_DISABLE_COUPON_REWARD
Force disable goPlay coupon reward

# PECAN_DEVELOPMENT
For Pecan development team, Use to enable debug script for testing. Must not use in production.
